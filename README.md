# react-nhost

To use Nhost `@nhost/nhost-js` with React this library provides a `NhostProvider` and some additional hooks.

## Development

- Install dependencies:

```bash
npm install
```

- Run the playground:

```bash
npm run play
```

- Run the unit tests:

```bash
npm run test
```

- Build the library:

```bash
npm run build
```
